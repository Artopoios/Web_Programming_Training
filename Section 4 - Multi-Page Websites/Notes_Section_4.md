Lesson 4.0 (25.) Computer File paths
-Absolute file path vs Relative file path-

Example of Absolute File Path:
C:/Project/Images/cat.png

Example of Relative File Path:
Images/cat.png

Special Characters:

../essay.docx
Χρησιμοποιείται για να πας ένα φάκελο προς τα έξω.

./dog.png
Μένει στον ίδιο φάκελο.
